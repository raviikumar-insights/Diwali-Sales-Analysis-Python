# Diwali-Sales-Analysis-Python

# 🪔 Diwali Sales Analysis

## 📌 Overview
This project analyzes Diwali sales data using Python to uncover valuable insights about customer behavior, product preferences, and sales trends. The goal is to help businesses make data-driven decisions during festive seasons.

---

## 📊 Objectives
- Understand customer demographics (age, gender, occupation, etc.)
- Identify top-selling product categories
- Analyze purchasing patterns during Diwali
- Discover high-revenue generating segments

---

## 🛠️ Technologies Used
- Python 🐍
- Pandas
- NumPy
- Matplotlib
- Jupyter Notebook

---

## 📁 Dataset
The dataset contains sales transaction details such as:
- User ID
- Gender
- Age Group
- Occupation
- Product Category
- Purchase Amount

*(Dataset source can be added here if available)*

---

## 📈 Key Insights
- Married women in the age group 26–35 are major buyers
- IT, Healthcare, and Aviation sectors contribute significantly
- Food, Clothing, and Electronics are top-selling categories
- Most purchases come from specific states/regions
